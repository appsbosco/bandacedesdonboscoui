# Design System

## Theme

Interfaz administrativa clara para uso prolongado en ambientes de oficina. Fondo gris azulado muy claro, superficies blancas matizadas y azul institucional reservado para selección, enlaces y acciones principales.

## Color

- Base: Slate de Tailwind para fondos, bordes y texto.
- Acción y selección: blue-600 / blue-700.
- Éxito: emerald-50 / emerald-700.
- Atención: amber-50 / amber-800.
- Error o rechazo: rose-50 / rose-700.
- Nunca depender únicamente del color; acompañar con texto o icono.

## Typography

Inter como tipografía de producto existente. Escala compacta: 12 px para metadatos, 14 px para contenido, 16–20 px para títulos. Números y nombres importantes usan peso 600 o 700.

## Components

- Controles de 40–44 px con borde slate-200 y foco blue-500.
- Badges compactos con texto explícito y un indicador visual.
- Botón primario azul; verde solo confirma una aprobación; rojo se reserva para rechazo o error.
- Contenedores principales con borde de 1 px y sombra mínima, sin tarjetas anidadas.
- Selección mediante fondo azul tenue, borde completo y texto, no mediante una franja lateral.

## Layout

Flujo de cola: encabezado compacto, navegación por vistas, filtros operativos, resumen lineal, tabla y panel de expediente. El panel permanece visible en escritorio; en pantallas menores el detalle se presenta dentro del flujo vertical.

## Motion

Transiciones de estado entre 150 y 200 ms. Sin movimiento decorativo; respetar preferencias de movimiento reducido.
